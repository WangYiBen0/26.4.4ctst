# T744752 XOR and Ugly Equation

## 题目描述

给定 $k,a$，保证 $0\le a<2^k$，求以下方程的非负整数解：

$$(x^2\oplus x)\bmod 2^k=a$$

或判断无解，**若有解，你需要输出最小的解**。

可以证明，若有解，则一定存在 $x<2^k$ 的解。

## 输入格式

**本题有多组测试数据**，第一行输入一个正整数 $T$，代表数据组数。

对于每组数据，输入一行一个正整数 $k$ 和一个非负整数 $a$。

## 输出格式

对于每组数据，先输出一行一个 `YES` 或 `NO`，表示是否有解。

若有解，再额外输出一行一个非负整数 $x$，代表方程的最小解。

## 输入输出样例 #1

### 输入 #1

```
4
5 6
10 10
4 2
3 1
```

### 输出 #1

```
YES
2
YES
3
YES
6
NO

```

## 说明/提示

对于所有数据，保证：

- $1\le T\le 10^5$；
- $1\le k\le 60$；
- $0\le a<2^k$。
