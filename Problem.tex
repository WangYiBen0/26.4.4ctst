\subsection{问题描述}

给定正整数 $k$ 和非负整数 $a$，求最小的非负整数 $x$ 使得 $0\le x<2^k$ 且 $x^2\oplus x\bmod{2^k}=a$。

从文件\filename{begin.in}中读入数据。

本题有多组测试数据，第一行输入一个正整数 $T$，代表数据组数。

对于每组数据，一行两个非负整数 $k,a$。

\subsection{输出格式}

输出到文件 \filename{begin.out} 中。

对于每组数据，先输出一行一个字符串 \texttt{YES} 或 \texttt{NO}，表示是否存在解。

若有解，再输出一行一个整数 $x$，代表答案。

\subsection{样例1输入}

\begin{lstlisting}

5 6
10 10
4 2
3 1
\end{lstlisting}

\subsection{样例1输出}

YES
3

NO
\end{lstlisting}

  \item $x\oplus y=\gcd(x,y)\le m$
\end{itemize}

Ralsei 正躺在床上看数学书。

上面写着：

\begin{quote}
按位异或（bitwise xor）是一个二元运算符，一般来说可以表示为 $A\oplus B=C$

这是一种简单又神奇的运算

$A\oplus B$ 意味着将二进制下的 $A$ 与 $B$ 每一位拿出来，分别进行不进位的加法

即 $0\oplus 0=0,0\oplus 1=1,1\oplus 0=1,1\oplus 1=0$

最终重新组合成为新数 $C$

比如 $2\oplus 11=(0010)_2\oplus (1011)_2=(1001)_2=9$

有很多神奇的性质，相信读者可以轻松注意到
\end{quote}

突然，他感应到了什么。

Kris 与 Susie 一同进入了一个暗世界。

Ralsei 顾不得洗漱，立即出发与 KS 会面。

但是，他突然发现 KS 所在的暗世界很远，一时半会到不了。

所幸，他想起看过的一本魔法书，写道：

\begin{quote}
亲爱的魔法师读者啊

如果你想去某个很远的地方

可以使用传送魔法哟

只要有挂念的人在那里

就可以通过如下的方法传送：

  \begin{quote}
每个人都有一个自己的号码

这个号码非常特殊，有 $2^n-1$ 位，由 $0~2^n-1$ 的整数组成

每一位都是不相同的

号码可以进行如下的变换：

    \begin{quote}
选择两个 $0~2^n-1$ 的不相同的整数 $i$ 和 $j$

当满足 $i$ 与 $j$ 的异或和不小于号码对应位置的值的异或和

即 $a_i\oplus a_j\le i\oplus j$，即可交换
    \end{quote}
只要你知道你和那个人的号码 $p$ 和 $q$

如果 $p$ 能通过不超过 $3.2\times 10^6$ 次上面的变换变成 $q$

那么你就能直接飞跃东非大裂巴传送至那个人身旁
  \end{quote}
\end{quote}

Ralsei 发现自己知道 Kris 和 Susie 和身旁一万个 $\text{N}_2$ 分子（你先别管是不是人）的 ID。

但是他数学太差了，不知道能传送到哪个 ID 身旁。

请你推测对于每个 ID ，可能或是不可能传送。

如果可能，输出一个变换的方案。

如果不可能，输出 $-1$。

\subsection{问题描述}

给定 $0\sim 2^n-1$ 的排列 $p,q$，下标从 $0$ 开始。

定义一次操作为：

- 选择不同的两个下标 $i,j$，使得 $p_i\oplus p_j\le i\oplus j$；
- 交换 $p_i,p_j$。

你要在不超过 $3.2\times 10^6$ 次操作内将 $p$ 变成 $q$，输出一组解。

\subsection{输入格式}

从文件\filename{night.in}中读入数据。

\textbf{本题有多组测试数据}，第一行输入一个正整数 $T$，代表数据组数。

对于每组数据：

\begin{itemize}
    \item 第一行输入一个正整数 $n$。
    \item 第二行输入 $2^n$ 个非负整数，代表排列 $p$；
    \item 第三行输入 $2^n$ 个非负整数，代表排列 $q$。
\end{itemize}

\subsection{输出格式}

输出到文件 \filename{night.out} 中。

对于每组数据：

\begin{itemize}
  \item 第一行输出一个非负整数 $m$，代表操作次数；
  \item 接下来 $m$ 行，每行输出两个不同的非负整数 $i,j$，代表这次操作的两个下标。
\end{itemize}

\subsection{问题描述}

给定长度为 $n$ 的非负整数序列 $a_1, \ldots, a_n$，求：

$$\prod_{i=1}^n\prod_{j=i+1}^n(a_i\oplus a_j)$$

答案对 $2^{64}$ 取模。

\subsection{输入格式}

从文件\filename{prophecy.in}中读入数据。

本题输入包含多组数据。第一行一个整数 $T$，表示数据组数。对于每组数据：

\begin{itemize}
    \item 第一行，一个整数 $n$。
    \item 第二行，$n$ 个整数 $a_1, \ldots, a_n$。
\end{itemize}

\subsection{输出格式}

输出到文件 \filename{prophecy.out} 中。

对于每组数据，输出一行一个数，表示答案对 $2^{64}$ 取模后的结果。
